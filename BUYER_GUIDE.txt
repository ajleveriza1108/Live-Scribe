LIVE SCRIBE v0.8.1 — BUYER GUIDE

WHAT LIVE SCRIBE DOES
---------------------
Live Scribe transcribes live microphone audio, computer audio, livestreams,
Zoom/Google Meet sessions, and recorded video or audio files. It runs locally
after one speech model has been downloaded.

No additional LLM is required for pause/resume, session recovery, topic
profiles, floating captions, transcript editing, speaker labels, markers,
session search, or storage cleanup.



STOPPING A MODEL DOWNLOAD
-------------------------
Stop Download safely keeps partial model files for resume. Live Scribe uses the
standard resumable Hugging Face transfer path so stopping does not produce an
hf-xet traceback in the launcher window.

Wait until the progress card closes before exiting the app or removing a USB
drive. Select the same speech quality later and download again to resume.

MODEL DOWNLOAD PROGRESS
-----------------------
The download has separate Downloading, Finalizing, and Ready stages.

The bar will not display 100% while Hugging Face is still completing or
verifying files. At the end of the transfer, the card changes to Finalizing and
uses an animated bar. Keep Live Scribe open until the ready message appears and
the progress card closes.

A resumed download may begin at a high percentage because partial files were
kept. Existing bytes are not counted as new download speed.


FIRST-RUN REPORT
----------------
The automatic PC compatibility report appears once for each portable copy.
Live Scribe saves a permanent local first-run marker before the rest of the
interface opens. The popup will not return automatically on later launches.

Use Models > Check This PC Again only when a new visible report is needed.

FIRST USE
---------
1. Keep the complete portable folder together.
2. Start Live Scribe through the platform launcher.
3. Review the local PC and portable-drive check.
4. Choose one buyer-friendly speech quality.
5. Download it once. A stopped download can be resumed later.
6. Choose the language and topic profile.


OPTIONAL NOISE REDUCTION
------------------------
Settings includes Light live noise reduction for transcription (optional). It is off by default and processes only audio sent to transcription. The original WAV remains unchanged. Use it for steady fan, air-conditioner, hum, or room hiss. Turn it off when quiet or distant speech becomes less clear.

The separate Reduce steady background noise during WAV verification option creates an enhanced copy and preserves the original WAV. These are conservative tools, not perfect noise cancellation. No additional AI model is required.


INTERVIEW MODE
--------------
Interview Mode prepares likely questions before an interview and privately
shows prepared or local-AI answer suggestions.

Create a profile with the target job, company, resume, job description,
experience, skills, projects, strengths, weaknesses, and answer preferences.
Choose Generate Interview Template and Question Bank.

Prepared answers are matched locally and can appear almost immediately after a
question is completed. Optional generated answers require a separately running
local llama.cpp-compatible server.

The transcript contains spoken words only. Private suggestions remain in the
Interview Assistant panel.

Version 0.8.0 labels one incoming audio stream at a time. Select Interviewer or
Interviewee before the next phrase. Simultaneous microphone and computer-audio
capture is planned separately and is not represented as complete here.

Use live assistance only when the interview rules allow it. Preparation and
mock practice are the safest primary uses.


INPUT AND TRANSCRIPT TOOLS
--------------------------
Test Input displays a live sound bar without starting transcription or saving
audio.

The complete microphone dropdown is clickable. Microphones that cannot
currently be opened are visible but disabled.

Summarize & Format creates an offline quick summary, key points, decisions,
action items, and a readable formatted transcript. The raw transcript remains
unchanged.

On Windows build 20348 or newer, Selected app audio can isolate a chosen
application after the Windows helper is installed. Listen to this app can be
turned off and on while the transcription session remains open. Other computer
sounds can continue playing without being included in the selected-app capture.

The selected-app option remains disabled until the repository's Windows helper
workflow has built and installed the native helper.

LIVE SESSION
------------
1. Enter a session title.
2. Choose Microphone or Computer audio / livestream.
3. Confirm that the audio meter shows a usable signal.
4. Click Start Listening.
5. Use Pause and Resume when needed.
6. Click Stop & Save WAV.
7. Click Verify from WAV for the separate full-recording pass.


CHOOSE VIDEO OR AUDIO FILE
--------------------------
This button remains clickable whenever Live Scribe is idle.

When no downloaded speech quality is ready, the button explains what is
required and offers to open Models. It is disabled only during another active
download, model load, live session, or transcription operation.

RECORDED VIDEO OR AUDIO
-----------------------
Click Transcribe File and choose a supported recording such as:

MP4, MKV, MOV, AVI, WebM, MP3, WAV, M4A, AAC, FLAC, OGG, OPUS, WMA,
AIFF, ALAC, or another format listed in the file chooser.

Video images are ignored. Live Scribe transcribes only the audio track.

TRANSCRIPT EDITOR
-----------------
Select a transcript line to:

- edit text
- set or remove a speaker
- play from its timestamp
- mark Important, Action Item, or Question
- mark the line as checked

Press F11 for floating captions. Ctrl+1, Ctrl+2, and Ctrl+3 can assign the
selected line to Speaker 1, Speaker 2, or Speaker 3.

SAVED SESSIONS
--------------
The Sessions page searches local saved sessions by title, transcript, topic,
speaker, language, and marker. Deleting a library entry does not delete the
original recording or exports.

RECOVERY
--------
If Live Scribe or the computer is interrupted, the next launch may offer to
recover an unfinished transcript and available WAV safety parts.

PORTABLE USE
------------
A fast USB 3.x flash drive or portable SSD is recommended. Keep the whole
folder together and never remove the drive during recording, model download,
transcription, verification, export, or saving.

EXPORTS
-------
DOCX, TXT, SRT, VTT, CSV, Markdown, and clipboard copy are available.

IMPORTANT ACCURACY NOTICE
-------------------------
Always verify names, numbers, dates, amounts, quotations, technical terms, and
other important information against the original WAV, video, or audio source.
