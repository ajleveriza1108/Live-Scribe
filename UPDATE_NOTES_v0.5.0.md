# Live Scribe v0.5.0

## Modern multilingual buyer interface

- Replaced the main interface with CustomTkinter.
- Added OLED Black and Dirty White themes.
- Added compact navigation for Live Session, Vocabulary, Models, and Settings.
- Made the transcript the visual focus of the Live Session page.
- Replaced technical model identifiers with Compact, Balanced, Best Overall, and Maximum Accuracy.
- Displayed approximate download sizes and plain-language guidance.
- Limited buyer model choices to four multilingual speech-quality options.
- Added English, Filipino/Tagalog, Taglish, Spanish, French, German, Italian, Portuguese, and Dutch modes.
- Preserved Auto Detect as an optional mode.
- Added a language-testing guide with eight YouTube samples.
- Preserved microphone capture, livestream/system-audio capture, WAV recording, manual Verify from WAV, vocabulary, and exports.

Grammar and diction suggestions remain intentionally limited to English, Filipino, and Taglish. Other languages still receive the full-WAV comparison and confidence-related review.
