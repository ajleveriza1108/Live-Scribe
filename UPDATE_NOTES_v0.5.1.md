# Live Scribe v0.5.1

## Stop and resume model downloads

- Added **Stop Download** to the temporary model progress card.
- Stopping preserves completed and partial files.
- Choosing the same speech quality and downloading again resumes the transfer.
- The Stop button changes to **Stopping…** while Live Scribe closes the active transfer safely.
- The app tells the buyer to wait until the progress card disappears before closing.

## Vocabulary Manager

- Added an explicit **Add New** action.
- Added **Save Changes** for editing spelling and pronunciation aliases.
- Added **Remove Selected** for deletion.
- Selecting a saved row enters edit mode and prevents accidental duplicate additions.
